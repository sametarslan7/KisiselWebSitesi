<h1 align="center">📁 Kişisel Portfolyo Web Sitesi</h1>

<p align="center">
  Bu proje, <b>ASP.NET MVC</b> ile geliştirilmiş bir kişisel portfolyo web sitesidir. Web sitesi, kişisel bilgilerinizi, projelerinizi, deneyimlerinizi ve sertifikalarınızı sergilemenize olanak tanır. Ayrıca, yönetim paneli ile içerik güncellenebilir, eklenebilir veya silinebilir.
</p>

<h2>🔗 Proje Bağlantıları</h2>

<ul>
  <li> <b>Canlı Demo:</b> <a href="https://sametarslancv.com" target="_blank">sametarslancv.com</a> </li>
  <li> <b>GitHub Reposu:</b> <a href="https://github.com/sametarslan7/KisiselWebSitesi" target="_blank">GitHub Repository</a> </li>
</ul>

<h2>🎯 Proje Amacı</h2>
<p>
  Bu proje, web geliştirme yeteneklerinizi ve çalışmalarınızı sergilemek, ayrıca admin paneli ile portfolyonuzu yönetilebilir hale getirmek amacıyla geliştirilmiştir. Kullanıcılar, web sitesi üzerinden kişisel bilgilerinizi, projelerinizi, deneyimlerinizi ve sertifikalarınızı görebilir ve iletişim formu aracılığıyla size ulaşabilirler. Aynı zamanda, admin paneli sayesinde bu bilgileri kolayca yönetebilirsiniz.
</p>

<h2>🚀 Özellikler</h2>

<ul>
  <li><b>Mobil uyumlu tasarım:</b> Tüm cihazlarda mükemmel görünüm.</li>
  <li><b>Projeler, Deneyimler ve Sertifikalar bölümü:</b> Web sitesindeki içerikler dinamik olarak veritabanından gelir ve admin paneli üzerinden yönetilir.</li>
  <li><b>Önyazı alanı:</b> Kişisel bir önyazı ekleyebilir ve yönetebilirsiniz.</li>
  <li><b>Admin paneli:</b> Kişisel bilgiler, projeler, deneyimler ve sertifikalar gibi içeriklerinizi giriş yaparak güncelleyebilir, ekleyebilir ve silebilirsiniz.</li>
  <li><b>Giriş yapma:</b> Admin paneline kullanıcı adı ve şifre ile giriş yapılır.</li>
  <li><b>İçerik yönetimi:</b> Admin panelinde içerik ekleyebilir, silebilir veya mevcut içerikleri güncelleyebilirsiniz.</li>
  <li><b>SEO dostu:</b> Arama motorları için optimize edilmiş yapı.</li>
  <li><b>Kolay yönetilebilir:</b> İçeriği hızlıca güncelleyebilmek için uygun yapı.</li>
</ul>

<h2>🔧 Kurulum ve Çalıştırma</h2>

<p>Projeyi kendi makinenizde çalıştırmak için aşağıdaki adımları izleyin:</p>

<ol>
  <li><b>Repo'yu klonlayın:</b>
    <pre>git clone https://github.com/kullaniciadi/portfolio.git</pre>
  </li>
  <li><b>Proje dosyasını açın:</b> Visual Studio veya Visual Studio Code ile açabilirsiniz.</li>
  <li><b>Bağımlılıkları yükleyin:</b> NuGet üzerinden gerekli paketleri indirin.</li>
  <li><b>Veritabanı bağlantısını ayarlayın:</b> <code>web.config</code> dosyasında <code>ConnectionString</code> kısmını kendi veritabanı bilgilerinizle güncelleyin.</li>
  <li><b>Projeyi başlatın:</b>
    <pre>dotnet run</pre>
  </li>
</ol>
